How do I get a key? (one-time, ~2 min)
  1. Go to github.com/settings/tokens?type=beta (Fine-grained tokens).
  2. Click Generate new token. Name it anything (e.g. "gallery").
  3. Under Repository access → Only select repositories → choose performance-gallery.
  4. Under Permissions → Repository permissions → set Contents to Read and write.
  5. Click Generate token, copy it, and paste it above. Press Save.
